# Opportunity Evaluation Scorecard

> **Quantitative** tool to compare multiple ideas.

## Scoring Dimensions
| Dimension | Weight | Description |
|-----------|--------|-------------|
| **Market Size** | 0.3 | TAM/SAM in monetary terms. |
| **Competitive Landscape** | 0.2 | Number of direct competitors, barrier to entry. |
| **Revenue Potential** | 0.25 | Projected annual revenue (Year 3). |
| **Execution Difficulty** | 0.15 | Technical & operational complexity. |
| **Strategic Fit** | 0.1 | Alignment with long‑term thesis. |

## Scoring Scale
- **1‑5** for each dimension (5 = excellent).  
- **Weighted Score** = rating × weight.  
- **Total Score** = sum of weighted scores (max = 5).
