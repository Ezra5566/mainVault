# Growth Systems

> **Framework** for systematic, repeatable user acquisition & retention.

## Core Loops
| Loop | Description | Metric | Target |
|------|-------------|--------|--------|
| **Acquisition Loop** | Visitor → Sign‑up → Paid | Conversion Rate | 10% |
| **Activation Loop** | First‑use → Value realization | Activation Rate | 70% |
| **Retention Loop** | Repeat purchase → Referral | Retention Rate (30‑day) | 40% |
| **Revenue Loop** | Upsell → Cross‑sell | Avg. Revenue per User (ARPU) | +20% QoQ |

## Funnel Stages
1. **Awareness** – content, SEO, ads.  
2. **Consideration** – webinars, free trials.  
3. **Conversion** – pricing page, checkout.  
4. **Retention** – email drip, loyalty program.
