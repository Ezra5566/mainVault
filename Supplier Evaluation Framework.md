# Supplier Evaluation Framework

> **Scorecard** to rank Chinese suppliers on strategic fit.

## Evaluation Criteria (Weight %)
| Criterion | Weight | Description |
|-----------|--------|-------------|
| **Cost Competitiveness** | 30% | Unit price + shipping. |
| **Quality Assurance** | 25% | ISO certifications, past defect rate. |
| **Lead Time** | 15% | Days from PO to port. |
| **Minimum Order Quantity (MOQ)** | 10% | Ability to start small. |
| **Communication Responsiveness** | 10% | Avg. response time (hours). |
| **Scalability** | 10% | Capacity to increase volume. |
| **Sustainability / Ethics** | 5% | Environmental certifications. |
| **Location Flexibility** | 5% | Proximity to ports / logistics hubs. |

## Scoring Process
1. Collect data for each supplier (documents, site visits).  
2. Assign a **0‑5** rating per criterion.  
3. Compute **weighted score** → rank suppliers.  
4. Conduct **sample audit** on top‑3.
