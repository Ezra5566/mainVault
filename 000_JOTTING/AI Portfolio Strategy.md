# AI Portfolio Strategy

> **Curate a showcase** of AI work that attracts investors, clients, and collaborators.

## Portfolio Components
1. **Personal Projects** – end‑to‑end AI services (e.g., demand‑forecasting API).  
2. **Open‑Source Contributions** – PRs to popular ML libraries.  
3. **Published Analyses** – blog posts, technical papers, case studies.  
4. **Product Launches** – MVPs that have paying users.  
5. **Metrics** – GitHub stars, API request volume, revenue generated.

## Presentation Templates
- **One‑pager** (problem → solution → tech stack → impact).  
- **Demo video** (≤ 3 min) hosted on YouTube/Drive.  
- **Metrics dashboard** (live Grafana or simple spreadsheet).

## next actions
- Choose **3 flagship projects** to feature by Q2 2027.  
- Build a **public GitHub repo** with README, CI badge, and demo link.  
- Create a **PDF portfolio** to attach to outreach emails.
