# Productized Services Framework

> Convert **custom AI services** into repeatable, scalable offerings.

## Steps
1. **Identify Core Value** – isolate the most repeatable task (e.g., “automated invoice classification”).  
2. **Package** – define a standard **feature set**, pricing, and SLA.  
3. **Automate Delivery** – scripts, Docker containers, API endpoints.  
4. **Marketing** – create landing page, case studies, demo videos.  
5. **Sales Funnel** – integrate with Lead Generation Playbook.  
6. **Support** – ticketing system, knowledge base (Obsidian).  

## Example Products
- **Invoice‑AI** – auto‑categorize vendor invoices (KSh 30 k/month).  
- **Chatbot‑Support** – multilingual FAQ bot for SMEs (KSh 150 k/month).  
- **Demand‑Forecast** – time‑series model for inventory (KSh 200 k/month).

## next actions
- Choose **one service** to productize (e.g., “invoice classification”).  
- Draft a **product spec** (features, pricing, tech stack).  
- Build a **MVP** in a separate repo; add CI pipeline for auto‑deploy.
