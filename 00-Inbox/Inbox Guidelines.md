# Inbox Processing Guide

## Purpose
The inbox is a temporary holding area for new ideas, tasks, and information that needs to be processed.

## Processing Steps
1. **Capture** - Add everything here without judgment
2. **Clarify** - Determine if actionable
3. **Organize** - Move to appropriate section
4. **Reflect** - Review regularly
5. **Engage** - Take action

## Tips
- Process inbox daily
- Use the 2-minute rule: if it takes less than 2 minutes, do it immediately
- Link to related notes using [[wiki-style links]]
