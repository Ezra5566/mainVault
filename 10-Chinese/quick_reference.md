## "How to say it" Quick Guide

### Asking someone's name:
- 你叫什么名字？(nǐ jiào shénme míngzi?)
- "What is your name?"

### Asking where from:
- 你是哪国人？(nǐ shì nǎ guó rén?)
- "Which country are you from?"

### Asking if you understand:
- 你懂吗？(nǐ dǒng ma?)
- "Do you understand?"

### Saying you don't understand:
- 我不懂。(wǒ bù dǒng)
- "I don't understand"

### Asking "How much?"
- 多少钱？(duōshao qián?)
- "How much does it cost?"

### Making a request:
- 我想要... (wǒ xiǎng yào...)
- "I would like..."

### Ordering food:
- 我要点... (wǒ yào diǎn...)
- "I would like to order..."

## Key Grammar Patterns

### Positive Statement:
```
Subject + Verb + Object
我 + 吃 + 饭
I + eat + rice
```

### Yes/No Question:
```
Statement + 吗
你 + 吃 + 饭 + 吗?
Do you eat rice?
```

### Answer Yes:
```
是 / 对
Yes / Correct
```

### Answer No:
```
不
No
```

### Negation:
```
不 + Verb
不 + 吃
don't eat
```
