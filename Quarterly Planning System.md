# Quarterly Planning System

> **3‑month cycle** that translates annual objectives into actionable tasks.

## Steps
1. **Review** previous quarter’s outcomes (KPIs, lessons).  
2. **Set Quarterly Objectives** (3‑5).  
3. **Define Key Results** (quantifiable outcomes).  
4. **Backlog Grooming** – prioritize tasks from [[SaaS Opportunity Tracker]] & [[Startup Ideas Database]].  
5. **Resource Planning** – allocate budget, assign owners, set deadlines.  
6. **Execution** – weekly sprint reviews, use [[Weekly CEO Review]] for updates.  
7. **Retrospect** – evaluate scorecard, adjust next quarter’s plan.

## Template Example (Q1 2027)
- **Objective:** Launch MVP of AI Inventory Optimizer.  
- **Key Results:**  
  - Build prototype (≤ 8 weeks).  
  - Acquire 5 pilot retailers.  
  - Generate KSh 150,000 pre‑sale revenue.  
- **Tasks:**  
  - Conduct market research (2 weeks).  
  - Develop data pipeline (3 weeks).  
  - Run pilot with retailers (2 weeks).
